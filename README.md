# ESE 5460: Principles of Deep Learning Final Project

## Files Overview

- **Milestone #1.ipynb**  
  Fetches the original dataset and performs initial downsampling for train/validation/test splits.

- **ideation.ipynb**  
  Scratch/experimental notebook used for early brainstorming and exploratory work.

- **image_analysis.ipynb**  
  Scans the dataset to identify and flag corrupted or unusable image files.

- **downsample.ipynb**  
  Further reduces the validation and test sets to approximately 5,000 samples each.

- **Milestone #2***  
  Contains baseline models, evaluations, and other deliverables for Milestone 2.
